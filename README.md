# MOSIM-Integration

This repository contains the MOSIM-Plugin for Unreal. To use the plugin, it must be cloned to %Your UE4 Project%\Plugins\UE4_MOSIM, for example using the following command: 
```
git clone git@github.com:dfki-asr/MMI-UnrealPlugin.git %Your UE4 Project%\Plugins\UE4_MOSIM.
```
